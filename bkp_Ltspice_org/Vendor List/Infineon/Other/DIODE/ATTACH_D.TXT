This list contains the various chip codes for all
Infineon Technologies RF diodes. To model a diode 
connect the chip model to the package equivalent 
 circuit given in the ~\package directory.

Please note that there are only a few chip models 
available by now. 
Please refer to s_diode.lib (includes package 
capacitances) for further models.

                                           J.P. Schaffer
