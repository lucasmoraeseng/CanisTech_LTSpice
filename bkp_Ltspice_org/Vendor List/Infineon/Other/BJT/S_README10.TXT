SPICE 2G6 LIBs:
===============

>>>>>  For SPICE 2G6 text blocks and package equivalent 
       circuits text blocks for use in microwave
       simulators please see subdirectory 
       \NL-BJT\ for RF-transistors
       \NL-DIODE\ for diodes

In the file S_AFBJT.LIB you will find spice parameters for the 
Infineon Technologies AF-transistors. Parameter extraction was
done on a typical device.

In the file S_DIODE.LIB you will find spice parameters 
including package capacitances for the Infineon Technologies 
Schottky, tuning and pn-diodes. ( For pure chips please refer
to   \NL-DIODE\CHIPS\ )

Spice circuits for digital transistors (bipolar transistors
with integrated base resistors) can be found in S_ditr.lib.

In the file S_RFBJT.LIB you will find SPICE parameters for the 
Infineon Technologies RF- and SIEGET-transistors.
